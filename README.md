# Bourouisse Piece-Dz | منصة قطع الغيار الذكية

هذا المشروع هو تطبيق ويب متطور لبيع وشراء قطع الغيار في الجزائر، مدمج بتقنيات الذكاء الاصطناعي للبحث والفلترة اللحظية.

## 📍 تعليمات التصدير (Export)
بما أن المساعد البرمجي لا يمكنه إنشاء ملفات ZIP مباشرة، يرجى اتباع الخطوات التالية:

1. **العثور على الزر:** ستجد زر **Export** أو **Export Code** في الزاوية العلوية اليمنى من شاشة **Firebase Studio**.
2. **التحميل:** اضغط على الزر وسيقوم المتصفح بتحميل المشروع كاملاً في ملف ZIP واحد.
3. **التشغيل المحلي:**
   - قم بفك ضغط الملف.
   - افتح المجلد في VS Code.
   - قم بتشغيل الأمر `npm install` في الـ Terminal.
   - ابدأ التطوير بـ `npm run dev`.

## 🚀 التقنيات المستخدمة
- **Next.js 15 (App Router)**
- **Tailwind CSS & Shadcn UI**
- **Firebase (Auth & Firestore)**
- **Google Genkit (AI Search Features)**

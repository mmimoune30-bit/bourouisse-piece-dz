
# Bourouisse Piece-Dz | مشروع منصة قطع الغيار

هذا المشروع جاهز تماماً للانتقال من بيئة Firebase Studio إلى بيئة التطوير المحلية.

## 📍 خريطة واجهة المستخدم (أين تجد زر التصدير؟)
بما أن المساعد البرمجي (AI) لا يمكنه إنشاء ملفات مضغوطة (ZIP) مباشرة، يرجى اتباع هذه الخريطة في واجهة المتصفح:

```
-------------------------------------------------------------------------------
[ Firebase Studio ]   اسم المشروع | Bourouisse      (Search)  [Share] [Export] 
-------------------------------------------------------------------------------
                                                                      ^ هنا بالضبط
```

### ماذا تفعل إذا لم يظهر الزر؟
1. **تحديث الصفحة (Refresh):** أحياناً تتوقف بعض عناصر الواجهة عن الظهور، التحديث يحل المشكلة غالباً.
2. **ضبط حجم الخط (Zoom Out):** اضغط على زر `Ctrl` و `-` في لوحة المفاتيح. في بعض الشاشات الصغيرة، يختفي الزر بسبب ضيق المساحة.
3. **وضعية العرض:** تأكد أنك في وضع "المحرر" (Editor) عبر الضغط على أي ملف في القائمة اليسرى.
4. **زر النقاط الثلاث:** في بعض الحالات، يظهر زر بجانب زر Share يحتوي على ثلاث نقاط `...` عند الضغط عليه ستجد خيار **Export Code**.

## 🚀 خطوات التشغيل المحلي (بعد تحميل الملف)
1. قم بفك ضغط ملف `ZIP` الذي قمت بتحميله.
2. افتح المجلد في برنامج VS Code أو أي محرر نصوص.
3. افتح الـ Terminal واكتب: `npm install` لتثبيت كافة المكتبات.
4. للبدء في التشغيل، اكتب: `npm run dev`.

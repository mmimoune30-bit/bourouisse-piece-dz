
# Bourouisse Piece-Dz | مشروع منصة قطع الغيار

هذا المشروع جاهز تماماً للانتقال من بيئة Firebase Studio إلى بيئة التطوير المحلية.

## 📥 كيفية الحصول على الكود (ZIP)
1. ابحث عن زر **"Export"** في الزاوية العلوية اليمنى من واجهة Firebase Studio.
2. إذا لم يظهر الزر:
   - قم بتحديث الصفحة (Refresh).
   - تأكد من تسجيل دخولك بنفس الحساب المالك للمشروع.
   - جرب متصفحاً آخر (مثل Chrome).

## 🚀 خطوات التشغيل المحلي
بعد تحميل ملف الـ ZIP وفك الضغط:
1. افتح المجلد في Terminal.
2. قم بتثبيت المكتبات: `npm install`.
3. قم بإنشاء ملف `.env.local` وأضف مفاتيح Firebase الخاصة بك (تجدها في إعدادات مشروعك في Firebase Console).
4. ابدأ التشغيل: `npm run dev`.


# Bourouisse Piece-Dz | مشروع منصة قطع الغيار

هذا المشروع جاهز تماماً للانتقال من بيئة Firebase Studio إلى بيئة التطوير المحلية.

## 📍 أين تجد زر التصدير (Export)؟
انظر إلى الشريط العلوي للمتصفح في واجهة Firebase Studio:
```
-------------------------------------------------------------------------------
[ Firebase Studio ]   اسم المشروع | Bourouisse      (Search)      [Share] [Export] 
-------------------------------------------------------------------------------
                                                                      ^ هنا بالضبط
```

## 📥 كيفية الحصول على الكود (ZIP)
1. ابحث عن زر **"Export"** في الزاوية العلوية اليمنى (كما هو موضح في الرسم أعلاه).
2. إذا لم يظهر الزر:
   - **تحديث الصفحة (Refresh):** أحياناً تتوقف الواجهة عن الاستجابة.
   - **تصغير حجم الخط (Zoom Out):** جرب الضغط على `Ctrl` مع `-` فقد يكون الزر مختفياً بسبب صغر حجم الشاشة.
   - **القائمة الجانبية:** تأكد من أنك لست في وضع "Preview" فقط؛ يجب أن تكون في وضع المحرر.

## 🚀 خطوات التشغيل المحلي (بعد تحميل الملف)
1. افتح المجلد في Terminal.
2. قم بتثبيت المكتبات: `npm install`.
3. قم بإنشاء ملف `.env.local` وأضف مفاتيح Firebase الخاصة بك.
4. ابدأ التشغيل: `npm run dev`.

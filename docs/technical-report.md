
# التقرير التقني لمشروع Bourouisse Piece-Dz

## 1. نظرة عامة على المشروع
منصة إلكترونية متخصصة في سوق قطع غيار السيارات في الجزائر، تربط بين البائعين والمشترين مع نظام بحث ذكي.

## 2. تعليمات التصدير (Export Instructions)
هذا المشروع مصمم ليعمل في بيئة **Firebase Studio**. عند الرغبة في الانتقال للتطوير المحلي أو النشر الإنتاجي:
- استخدم زر **Export** في أعلى يمين الواجهة للحصول على ملف ZIP.
- المشروع يعتمد على **Next.js 15** و **Firebase Client SDK**.

## 3. الهيكل البرمجي (Tech Stack)
- **Frontend**: Next.js 15 (App Router)
- **Backend**: Firebase (Firestore, Auth)
- **AI**: Google Genkit (Gemini 2.5 Flash)
- **Styling**: Tailwind CSS + Shadcn UI

## 4. المكونات الأساسية
- `Navbar`: يحتوي على الشعار المتموج، شعار العجلة الدوارة، ومحرك البحث الذكي.
- `AISearchBox`: محرك البحث الموجود في الشريط الرمادي أسفل الشريط الأزرق.
- `Catalog`: نظام الفلترة الفورية بمجرد الكتابة.

## 5. حالة المشروع
المشروع جاهز تماماً للتصدير (Ready for Export) مع كافة ملفات الإعداد الضرورية مثل `firebase.json` و `.firebaserc`.
